FULLCONTACT API CLIENT

1. Retrieve your API key from FullContact's Developer Portal and paste it in the box below.
2. (Optional) Press the "Save Key" button to save the API key locally.
3. Click "Select CSV" and pick a CSV with names arranged either all in a single row or all in their own row.
4. Click "Perform Request" to perform a batch request to FullContact on all users included in the CSV.
5. Once you see data in the above textbox, click "Parse And Download CSVs" to download the CSVs to your Downloads folder.
WARNING: All existing CSV files will be overwritten without warning if they exist within the Downloads folder. Clicking the "Parse and Download CSVs" button will download the files with no additional feedback.
CSVs that will be downloaded:
  • photos.csv
  • contact_info.csv
  • organizations.csv
  • demographics.csv
  • social_profiles.csv
  • metadata.csv
  • digital_footprint_topics.csv
  • digital_footprint_scores.csv
